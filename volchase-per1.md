# VolChase — Brand Perspective 1
> Version 1.0 | June 2026
> Direction: **Cedehub Visual DNA × VolChase Identity**

---

## Overview

Per 1 borrows the entire visual language of Cede Hub — dark, minimal, cold, institutional — and injects the VolChase message and competitive spirit into it. The result: a platform that looks established and credible on first glance, but speaks like a race.

**The hypothesis:** If VolChase launched today wearing Cede Hub's clothes, would traders trust it immediately? Yes. Would they stay for the competition? That's the message's job.

## 2. Color Palette

Taken directly from Cede Hub's CSS — no deviation.

| Role | Hex | Usage |
|---|---|---|
| **Background** | `#0A0A0A` | Page canvas, app shell |
| **Surface** | `#111111` | Cards, modals, nav |
| **Surface Raised** | `#1A1A1A` | Hover states, active panels |
| **Primary** | `#FFFFFF` | Headlines, primary text |
| **Secondary Text** | `#888888` | Labels, meta, captions |
| **Accent** | `#6C3BFF` | CTAs, active states, highlights |
| **Accent Hover** | `#5A2EE0` | Button hover |
| **Border** | `#222222` | Dividers, card outlines |
| **Success** | `#22C55E` | Positive numbers, rewards confirmed |
| **Muted** | `#444444` | Disabled states |

### Color Logic
Cold and institutional. The accent purple does all the work — one color, used sparingly. Everything else is greyscale. This is Cede Hub's exact approach: let the dark canvas breathe, let numbers speak, let the accent mark the action.

## 3. Typography Direction

Following cedehub's Webflow-based type system.

| Role | Treatment |
|---|---|
| **Display / Hero** | Large, bold, geometric sans — uppercase where impact needed |
| **Section heads** | Medium weight, tracking slightly open |
| **Body** | Regular weight, 16px base, generous line height (1.6) |
| **Data / Numbers** | Monospace or tabular figures — leaderboard ranks, volume, rewards |
| **Labels / Meta** | Small caps or 12px uppercase, muted color |

**Font stack (cedehub approach):** System sans-serif stack with Webflow defaults — Inter or equivalent geometric sans. No serifs. No display scripts.

## 4. Visual Language

### Layout
- Full-width dark sections
- Max-width container ~1200px, centered
- Section padding: generous vertical (80–120px)
- Cards: subtle border `#222`, no drop shadows, borderless feel
- Grid: clean 12-col, content-first

### Components
- **CTA Buttons:** Filled accent purple, no border-radius or minimal (4px), medium weight label
- **Secondary Buttons:** Ghost style — border only, white text
- **Leaderboard rows:** Alternating subtle background, rank number left-aligned monospace
- **Campaign cards:** Dark surface, single accent stat, minimal decoration
- **Nav:** Fixed top, transparent → solid on scroll, logo left, links center/right

### Motion
Minimal — matching cedehub's near-static feel:
- Subtle fade-in on scroll
- No parallax
- Leaderboard rank changes: simple number tick animation
- CTA hover: slight brightness lift only

### What to avoid
- Gradients (cedehub uses none)
- Illustrations or characters
- Rounded cards (keep sharp or minimal radius)
- Emoji in UI (only in social copy)
- Any warm colors

## 5. Tone of Voice (Per 1 application)

Per 1 keeps VolChase's voice but delivers it at **cedehub's volume** — quieter, more restrained than the full KoinRace energy. The words are competitive; the delivery is calm.

| Full KoinRace voice | Per 1 (cedehub-toned) |
|---|---|
| "$5,000 up for grabs. Race starts NOW." | "$5,000 prize pool. 48 hours. Top 10 win." |
| "Rank 1 is PULLING AWAY 🔥" | "Rank 1 is pulling away. 18 hours left." |
| "Volume doesn't lie." | "Volume = rank. Transparent, on-chain, verifiable." |

The message is the same. The energy is turned down one notch to match the visual restraint.

## 7. Differentiation from Cede Hub

Per 1 looks like Cede Hub. But here's what's different:

| Element | Cede Hub | VolChase Per 1 |
|---|---|---|
| **Name** | Cede Hub | VolChase |
| **Tagline** | Trade \| Hold \| Earn | Turn Volume Into Victory |
| **Hero message** | Generic benefit | Competition-first framing |
| **Voice** | Flat, no personality | Restrained but directional |
| **CTA** | "Earn now" | "Race now" |
| **Brand concept** | Utility platform | Competition platform wearing utility clothes |

The visual is borrowed. The identity is not.
